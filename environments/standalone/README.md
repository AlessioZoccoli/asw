# Standalone 

Questo è l'ambiente di esecuzione più semplice, 
composto da una sola macchina virtuale **node**. 

## Descrizione delle macchine virtuali 

### node

La macchina virtuale **node** ha il seguente software 

* Ubuntu Trusty (14.04 LTS) a 64 bit 

* Oracle Java SDK 
  
Configurazione di rete 

* Indirizzo IP: 10.11.1.251 

* Porte pubblicate sull'host: nessuna
